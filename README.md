### Run Locally

To run **Filmlane** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/filmlane.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/filmlane.git
```
